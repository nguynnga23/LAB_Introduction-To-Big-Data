I. PostgreSQL

Tương tự như SQL Sever.

1. Cài đặt PostgreSQL với docker compose

services:
  postgres:
    image: postgres:13
    environment:
      POSTGRES_USER: lab8
      POSTGRES_PASSWORD: lab8
      POSTGRES_DB: lab8
    healthcheck:
      test: ["CMD", "pg_isready", "-U", "airflow"]
      interval: 10s
      retries: 5
      start_period: 5s
    ports:
      - "5432:5432"
    expose:
      - 5432

II. MongoDB

1. Cài đặt docker compose

services:
  mongodb:
    image: mongo:4.0.8
    container_name: mongodb
    command: mongod --auth
    environment:
      MONGO_INITDB_ROOT_USERNAME: admin
      MONGO_INITDB_ROOT_PASSWORD: admin
      MONGO_INITDB_DATABASE: lab8
      MONGODB_DATA_DIR: /data/db
      MONDODB_LOG_DIR: /dev/null
    ports:
      - "27017:27017"

2. Kết nối MongoDB

Sử dụng MongoDB CLI

docker exec -it <container_id> mongosh -u <user> -p <password>

Hiển thị toàn bộ databse:

show dbs

Xem database hiện tại

db

Chuyển sang db khác - tạo db mới

use <db_name>

Xem toàn bộ collection

show collections

Thực hiện câu truy vấn - thêm

db.<collection_names>.insertOne(<document>)

Thực hiện câu truy vấn

db.<collection_names>.find({<condition>})
db.<collection_names>.findOne({<condition>})

Bài tập

Với dữ liệu được cho trước, hãy thực hiện:

  • Tạo database với tên của mình
  • Tạo 2 bảng (postgress ), collection (mongodb)
  • Insert dữ liệu vào 2 bảng và collection
  • Viết câu query (postgress và mongodb) để trả lời câu hỏi:
    • Có bao nhiêu bạn vắng thi môn Toán (không có điểm)
    • Bao nhiêu bạn thi môn “Vật lý” đạt điểm giỏi.

III. Airflow

1. Cài đặt airflow với docker compose

Document: https://airflow.apache.org/

Hướng dẫn cài đặt airflow với docker compose

Các thành phần cơ bản của Airflow

DAG (Directed Acyclic Graph)

  • bash_operator
  • python_operator

Ví dụ: Tạo workflow thực thi mỗi 30s bao gồm:

  • Sử dụng bash_operator in ra màn hình nội dung “Task start”
  • Sử dụng python_operator in ra màn hình nội dung “Task end”

import datetime as dt
from airflow import DAG
from airflow.operators.bash_operator import BashOperator
from airflow.operators.python_operator import PythonOperator


default_args = {
    'owner': 'Lab8-example',
    'start_date': dt.datetime.now() - dt.timedelta(minutes=1),
    'retries': 1,
    'retry_delay': dt.timedelta(seconds=30),
}


def printEnd():
    print("Task end")

with DAG('Lab8_example',
         default_args=default_args,
         tags=['LAB8'],
         schedule_interval=dt.timedelta(minutes=5),      # '0 * * * *',
         ) as dag:

    task1 = BashOperator(task_id='task1',
                       bash_command='echo "Task start"')

    task2 = PythonOperator(task_id='task2',
                            python_callable=printEnd)
    

task1 >> task2

Copy file trên vào thư mục dags

Truy cập vào container ‘scheduler’ thực thi lệnh

airflow scheduler
